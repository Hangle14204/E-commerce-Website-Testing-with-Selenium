# E-commerce Website Testing with Selenium

## Overview

This project focuses on testing an e-commerce system, including both user and admin functionalities. It combines manual test design and automated testing using Selenium WebDriver.

## Scope of Testing

* User: Register, Login, Search, View Product
* Cart: Add, Update, Remove items
* Checkout: Place order
* Admin: User management, Product management

## Testing Activities

* Participated in test planning and designed 70 test cases covering system functionalities and UI
* Developed 56 automated test scripts using Selenium WebDriver with NUnit (C#)
* Executed tests, identified and reported 26 defects
* Tracked bug fixing progress using Jira and performed re-testing

## Technologies Used

* Language: C#
* Automation: Selenium WebDriver
* Test Framework: NUnit
* Tool: Visual Studio
* Bug Tracking: Jira

## How to Run

1. Open the solution in Visual Studio
2. Build the project
3. Run test cases

## Test Environment
- Target system: Local e-commerce website
- URL: https://localhost:xxxx

## Note
The tested website runs locally and is not included in this repository.
Test scripts are provided for demonstration purposes only.
